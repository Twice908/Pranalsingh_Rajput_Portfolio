# Pranalsingh Rajput - Portfolio

A standalone HTML portfolio site, deployed automatically to GitHub Pages via GitHub Actions on every push to `main`.

Live at: `https://<username>.github.io/<repo-name>/`

## Repo structure

- `HTML/pranalsingh-twice-portfolio.html` - the portfolio page (deployed as `index.html`)
- `FavICon/favicon.svg` - animated favicon (deployed as `favicon.svg`)
- `docs/` - resume PDFs (deployed to site root for download)
- `.github/workflows/deploy.yml` - CI/CD pipeline that builds and deploys to GitHub Pages
